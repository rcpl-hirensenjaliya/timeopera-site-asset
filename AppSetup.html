Setup Guide

TimeOpera App – Prerequisites & Setup Guide

Follow these straightforward steps to deploy and configure TimeOpera in your Microsoft 365 tenant. The setup covers deploying the app package, approving Graph API permissions, completing onboarding, and integrating with your SharePoint site.

1. Upload Solution to App Catalog

  • Request the TimeOpera .sppkg file from our team
  • Navigate to SharePoint Admin Center → App Catalog → Apps for SharePoint
  • Upload the .sppkg file
  • Click Deploy and make it available to all sites (if required)

2. Approve API Permissions (Microsoft Graph)

  • Go to SharePoint Admin Center → API Access
  • Review the pending permission requests
  • Approve all required Microsoft Graph permissions
⚠️ Admin approval is mandatory for the application to function properly

3. Complete Application Onboarding

  • Open the Onboarding link provided by the team
  • Fill in all required configuration details
  • Submit the form to complete the setup

4. Configure Shared Mailbox

  • Create or use an existing Shared Mailbox in Microsoft 365
  • Assign required access permissions to users/app
  • This mailbox will be used for application communication

5. SharePoint Site Setup

  • Ensure the target SharePoint site is available
  • Add required users and permissions
  • Verify that the TimeOpera app is accessible on the site

6. Final Validation

  • Verify the app is successfully deployed
  • Ensure all permissions are approved
  • Confirm onboarding is completed
  • Validate mailbox and SharePoint integration
